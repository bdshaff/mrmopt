# Average and marginal return curves with current and optimal points

Produces faceted average return (AR) and marginal return (MR) curves for
each channel, with points marking the current and optimal spend
positions.

## Usage

``` r
opt_plot_returns(x, ...)
```

## Arguments

- x:

  An \`opt_mix_result\` object returned by \[opt_mix()\].

- ...:

  Additional arguments (currently unused).

## Value

A ggplot object.
