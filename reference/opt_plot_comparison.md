# Dumbbell chart of spend reallocation

Produces a dumbbell/segment chart showing the shift from current to
optimal weekly spend per channel.

## Usage

``` r
opt_plot_comparison(x, ...)
```

## Arguments

- x:

  An \`opt_mix_result\` object returned by \[opt_mix()\].

- ...:

  Additional arguments (currently unused).

## Value

A ggplot object.
