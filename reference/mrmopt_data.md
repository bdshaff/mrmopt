# Simulated multi-channel media spend dataset

A simulated dataset containing weekly media spend and conversions for
five advertising channels over two years (104 weeks). The data are
generated from known response curves (Gompertz, Logistic, and
Log-Logistic) with additive Gaussian noise, and are intended for use in
package examples and vignettes.

## Usage

``` r
mrmopt_data
```

## Format

A tibble with 520 rows and 4 columns:

- channel:

  Factor. Advertising channel name. One of `"Paid Search"`,
  `"Paid Social"`, `"Display"`, `"Online Video"`, `"TV"`.

- week:

  Date. Week start date (Monday), ranging from 2023-01-02 to 2024-12-23.

- spend:

  Integer. Weekly media spend in US dollars.

- conversions:

  Integer. Weekly conversions (KPI).

## Details

Each channel has a distinct spend scale and response curve shape:

- **Paid Search**: Gompertz curve, ~\$45k/week average spend.

- **Paid Social**: Logistic curve, ~\$27k/week average spend.

- **Display**: Gompertz curve, ~\$18k/week average spend.

- **Online Video**: Logistic curve, ~\$34k/week average spend.

- **TV**: Log-Logistic curve, ~\$80k/week average spend.

The dataset is generated by `data-raw/mrmopt_data.R`.
